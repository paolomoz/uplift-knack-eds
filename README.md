# Snowflake
Create truly bespoke sites and pages for Edge Delivery Services.

This project is _optimized_ for Stardust prototypes.

## Getting started

### 1. Github
1. Use this template to make a new repo.
1. Install [AEM Code Sync](https://da.live/bot).
1. Install the latest CLI: `sudo npm install -g @adobe/aem-cli`

### 2. Import
1. Place your startdust prototypes in `/stardust/prototypes`
2. Tell Claude you want to convert your stardust prototypes to edge delivery using the snowflake skill.
3. Follow the steps.

### 3. Run / Test
1. Ensure you have a `/content` folder.
1. Open your site: `aem up`.
